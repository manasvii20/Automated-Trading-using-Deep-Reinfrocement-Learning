Proposed Project Title:
Automated Trading Using Deep Reinforcement Learning (FinRL Framework)

Project Abstract:
This project focuses on developing an automated trading system using deep reinforcement learning techniques. The project will be implemented using the FinRL open-source framework, which provides a complete pipeline for financial reinforcement learning, including data preprocessing, market environment simulation, and deep reinforcement learning agents. The system enables an AI agent to learn optimal trading actions such as buy, sell, or hold by interacting with historical financial market data and maximizing long-term rewards.
As part of this group project, we will study the FinRL architecture, implement deep reinforcement learning models for automated trading, and evaluate their performance using real-world market data. The objective is to understand the practical application of reinforcement learning in financial markets and assess the effectiveness of AI-driven trading strategies in terms of profitability and risk management.
